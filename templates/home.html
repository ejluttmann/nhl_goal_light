NHL GOAL LIGHT !!

API DOCUMENTATION

/api/v1/season Return true if a Hockey season is in progress

/api/v1/teams : Get the list of teams

/api/v1/team/<team>/id : Get the ID of a team, ex: /api/v1/team/Canadiens/id

/api/v1/team/<team>/game : Return true if a game is in progress for the given team. Ex: /api/v1/team/Canadiens/game

/api/v1/team/<team>/score : Get the score of a team, ex: /api/v1/team/Canadiens/score

/api/v1/team/<team>/watch_goals : Follow the current game of the specified team,
and activate the goal light if a goal is scored, ex: /api/v1/team/Canadiens/watch_goals

/api/v1/team/<team>/watch_goals?delay=5 : Follow the current game of the specified team,
and activate the goal light if a goal is scored, ex: /api/v1/team/Canadiens/watch_goals?delay=5

/api/v1/goal_light/activate : Activate the Goal light
